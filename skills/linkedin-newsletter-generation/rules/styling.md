# Visual Styling Guidelines v1.1

(Optimized for Strategy/Finance/Consulting Thought Leadership)

Blue communicates trust, reliability, stability, and professionalism. Minimalist aesthetic focuses attention on intellectual content.

---

## Brand Colors

### Primary Palette (60-70%)
| Color | Hex | Purpose |
|-------|-----|---------|
| Deep Navy | `#0d1b2a` | Primary backgrounds, authority |
| Navy Blue | `#1a365d` | Headers, key elements |
| Slate Blue | `#2c5282` | Secondary elements, gradients |

### Neutral Palette (20-30%)
| Color | Hex | Purpose |
|-------|-----|---------|
| Charcoal | `#1a202c` | Body text on light |
| Cool Gray | `#4a5568` | Secondary text |
| Light Steel | `#a0aec0` | Borders, dividers |
| Off-White | `#f7fafc` | Text on dark, backgrounds |

### Accent (sparingly, <10%)
| Color | Hex | Purpose |
|-------|-----|---------|
| Forest Green | `#276749` | Subtle wealth/growth signal |

### Colors to Avoid
- Red (associated with "being in the red")
- Orange (signals creativity over analytical rigor)
- Yellow (casual or warning-like)
- Bright/saturated colors (undermine professional credibility)

---

## Typography

### Font Stack
```
--font-primary: 'Inter', 'Roboto', -apple-system, BlinkMacSystemFont, sans-serif;
```

### Weight Hierarchy
| Element | Weight | Size |
|---------|--------|------|
| Headlines | 600-700 | 24-32px |
| Body | 400 | 16-18px |
| Captions | 400 | 12-14px |

### Rules
- Line height: 1.5-1.6 for body
- Max line width: 65-75 characters
- Use weight contrast, not size alone
- Avoid all-caps except short labels

---

## Image Generation Guidelines

### Core Principles
1. **Abstract over literal**: Geometric patterns, not stock photos
2. **Avoid photorealism**: Stylized, glass-morphism, wireframe
3. **Navy dominance**: Colors from primary palette only
4. **Minimalist composition**: Clean, uncluttered, intentional negative space

### Mood Keywords
- Calm authority
- Intellectual rigor
- Strategic clarity
- Premium, understated
- Thinking model, not advertisement

### Prompt Template
```
Abstract geometric composition with deep navy (#0d1b2a) and navy blue (#1a365d) gradient. Minimal professional design featuring [ABSTRACTED OBJECT/METAPHOR]. Clean, sophisticated corporate aesthetic. No people, no text, no faces. Subtle depth with layered translucent geometric elements. Premium, understated mood conveying intellectual rigor.
```

### Theme Elements by Topic
| Post Topic | Visual Element |
|------------|----------------|
| Strategy/Trade-offs | Intersecting geometric planes, decision branches |
| Capital/Finance | Flowing abstract patterns, layered depth |
| Incentives/Mechanisms | Interconnected network lines, subtle nodes |
| Uncertainty/Risk | Gradient transitions, asymmetric forms |
| Institutional | Structured geometric boundaries |
| CS/Software Engineering | Modular grids, architectural layers |
| Artificial Intelligence | Probabilistic gradients, high-dimensional nodes |
| Entrepreneurship | Iteration cycles, resource allocation blocks |

## LinkedIn Visual Identity

### Profile Standards
- Headshot: Professional, well-lit, neutral background
- Banner: Abstract navy geometric pattern or gradient
- Post visuals: Generated abstract images from this guide

### What to Avoid
- Stock photos with humans
- Cluttered compositions
- Text overlays on images
- Bright colors competing with content
- Anything looking like marketing material

---

## Design Philosophy

The visual identity should feel like an **internal memo, not an advertisement**. Visual restraint signals that authority comes from reasoning quality, not visual polish.
